# Getting Started

The server scans Markdown files, rewrites links, and rebuilds the site whenever source files change.

Go back to [[Welcome to markdown2web]].

## Editing flow

1. Log in to `/admin`
2. Create or update notes
3. Save changes
4. Watch the site update automatically
