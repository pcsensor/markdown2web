### 一、准备工作
1. 登录机场后台，复制 **Clash 订阅链接**（推荐带 `flag=clash` 或 `flag=meta` 的）。
2. 确认服务器架构（一般是 `x86_64` 或 `aarch64`）。

### 二、推荐安装方式（两种，任选）

#### 方式 A：一键脚本（最推荐，省心）
使用社区维护很好的 `clash-for-linux-install`（支持 mihomo）：

```bash
# 先安装依赖
sudo apt update && sudo apt install -y curl tar unzip xz-utils procps git

# 一键安装（默认用 mihomo）
git clone --branch master --depth 1 https://gh-proxy.org/https://github.com/nelvko/clash-for-linux-install.git
cd clash-for-linux-install
bash install.sh mihomo
```

安装完成后会出现 `clashctl` 命令。  
导入订阅：

```bash
clashsub add "你的机场Clash订阅链接"
clashon          # 启动代理
clashctl status  # 查看状态
```

常用命令：
- `clashon` / `clashoff`：开关代理
- `clashsub update`：更新订阅
- `clashui`：打开 Web 面板
- `clashtun on`：开启 TUN 全局透明代理（推荐）
